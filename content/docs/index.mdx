## Quick Start

CypherCN is a trusted registry for shadcn/ui. You can start adding components right away!

```bash
pnpm dlx shadcn@latest add @cyphercn/button
```

This will add the CypherCN registry to your `components.json`:

```json
{
  "registries": {
    "@cyphercn": "https://cyphercn.com/r/{name}.json"
  }
}
```

### Initialize MCP (optional)

This adds configuration to your IDE, enabling you to use the components directly in your code with AI assistance. The AI will have full context of the entire CypherCN library.

```bash
pnpm dlx shadcn@latest mcp init
```
