## Usage

---

CypherCN sidebar is using Shadcn sidebar component. All you need to do is replace your app sidebar with CypherCN sidebar.

For more information, check out the [Shadcn Sidebar documentation](https://ui.shadcn.com/docs/components/sidebar).

### In your layout file:

```tsx
import { SidebarProvider, SidebarTrigger } from "@/components/ui/sidebar"
import { AppSidebar } from "@/components/ui/cypher/blocks/sidebar"

export default function Layout({ children }: { children: React.ReactNode }) {
  return (
    <SidebarProvider>
      <AppSidebar />
      <main>
        {children}
      </main>
    </SidebarProvider>
  )
}
```
