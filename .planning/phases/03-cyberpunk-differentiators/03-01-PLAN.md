Create a custom hook that manages command history state:

```typescript
interface UseCommandHistoryOptions {
  maxHistory?: number;        // Default 100
  persistKey?: string;        // localStorage key for persistence (optional)
}

interface UseCommandHistoryReturn {
  history: string[];
  historyIndex: number;
  currentValue: string;
  setCurrentValue: (value: string) => void;
  navigateUp: () => void;     // ArrowUp - go to older command
  navigateDown: () => void;   // ArrowDown - go to newer command
  submit: (value: string) => void;  // Add to history, reset index
  clear: () => void;          // Clear history
}
```

Implementation details:
- Store history array in state (newest at end, oldest at index 0)
- historyIndex starts at -1 (not navigating), increments for older commands
- navigateUp: if index < history.length - 1, increment and set currentValue
- navigateDown: if index > 0, decrement; if index === 0, reset to -1 and clear currentValue
- submit: append to history, reset index to -1, clear currentValue
- Optional localStorage persistence if persistKey provided
- Cap history at maxHistory (default 100), remove oldest when exceeded
