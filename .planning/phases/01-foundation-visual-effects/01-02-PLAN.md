```css
/* ─────────────────────────────────────────────────────────────────────────────
   CRT Screen Container (curvature + vignette)
   ───────────────────────────────────────────────────────────────────────────── */

.crt-screen {
  position: relative;
  overflow: hidden;
  /* Subtle barrel distortion via border-radius on pseudo-element */
  border-radius: 8px;
}

.crt-screen::before {
  content: "";
  position: absolute;
  inset: 0;
  /* Vignette effect - darker edges */
  background: radial-gradient(
    ellipse at center,
    transparent 50%,
    rgba(0, 0, 0, 0.15) 80%,
    rgba(0, 0, 0, 0.4) 100%
  );
  pointer-events: none;
  z-index: 20;
  border-radius: inherit;
}

/* Stronger curvature variant for dramatic effect */
.crt-screen-curved {
  position: relative;
  overflow: hidden;
  border-radius: 16px;
}

.crt-screen-curved::before {
  content: "";
  position: absolute;
  inset: -2%;
  /* Stronger vignette for curved appearance */
  background: radial-gradient(
    ellipse 120% 120% at center,
    transparent 40%,
    rgba(0, 0, 0, 0.2) 70%,
    rgba(0, 0, 0, 0.5) 100%
  );
  pointer-events: none;
  z-index: 20;
}

.crt-screen-curved::after {
  content: "";
  position: absolute;
  inset: 0;
  /* Subtle highlight on top edge for glass reflection */
  background: linear-gradient(
    180deg,
    rgba(255, 255, 255, 0.03) 0%,
    transparent 30%
  );
  pointer-events: none;
  z-index: 21;
  border-radius: inherit;
}

/* Combined scanlines + screen for full CRT effect */
.crt-full {
  position: relative;
  overflow: hidden;
  border-radius: 12px;
}

.crt-full::before {
  content: "";
  position: absolute;
  inset: 0;
  /* Scanlines */
  background: repeating-linear-gradient(
    0deg,
    transparent,
    transparent 1px,
    rgba(0, 0, 0, 0.12) 1px,
    rgba(0, 0, 0, 0.12) 2px
  );
  pointer-events: none;
  z-index: 10;
}

.crt-full::after {
  content: "";
  position: absolute;
  inset: 0;
  /* Vignette */
  background: radial-gradient(
    ellipse at center,
    transparent 50%,
    rgba(0, 0, 0, 0.15) 80%,
    rgba(0, 0, 0, 0.35) 100%
  );
  pointer-events: none;
  z-index: 20;
  border-radius: inherit;
}
```

Key design decisions:
- Uses `inset: 0` for full coverage (not top/left/right/bottom)
- `pointer-events: none` prevents blocking interactions (critical!)
- `z-index` ordering: content (default) < scanlines (10) < vignette (20) < reflection (21)
- `border-radius: inherit` ensures rounded corners propagate
- Position relative on parent for pseudo-element positioning

Update the phosphor glow classes to use CSS custom properties so they respond to color variants. Currently they hardcode `rgba(51, 255, 51, ...)` which only works for green.

Replace the existing phosphor glow section (lines ~62-82) with:

```css
/* ─────────────────────────────────────────────────────────────────────────────
   Phosphor Glow Effects (theme-aware)
   ───────────────────────────────────────────────────────────────────────────── */

/* Text glow - uses currentColor for theme awareness */
.phosphor-glow {
  text-shadow:
    0 0 2px currentColor,
    0 0 4px currentColor,
    0 0 8px var(--phosphor-glow, currentColor);
}

.phosphor-glow-subtle {
  text-shadow:
    0 0 1px currentColor,
    0 0 2px var(--phosphor-glow, currentColor);
}

.phosphor-glow-intense {
  text-shadow:
    0 0 2px currentColor,
    0 0 6px currentColor,
    0 0 12px var(--phosphor-glow, currentColor),
    0 0 20px var(--phosphor-glow, currentColor);
}

/* Border/box glow */
.phosphor-border-glow {
  box-shadow:
    0 0 1px currentColor,
    0 0 2px var(--phosphor-glow, currentColor),
    inset 0 0 1px var(--phosphor-glow, currentColor);
}

.phosphor-border-glow-intense {
  box-shadow:
    0 0 2px currentColor,
    0 0 4px var(--phosphor-glow, currentColor),
    0 0 8px var(--phosphor-glow, currentColor),
    inset 0 0 2px var(--phosphor-glow, currentColor);
}

/* Box shadow glow (for cards, panels) */
.phosphor-box-glow {
  box-shadow:
    0 0 4px var(--phosphor-glow, currentColor),
    0 0 8px var(--phosphor-glow, currentColor);
}
```

Key changes:
- Uses `currentColor` for the solid inner shadows (inherits text color)
- Uses `var(--phosphor-glow, currentColor)` for the diffuse outer glow
- Fallback to currentColor if --phosphor-glow isn't defined
- Added .phosphor-glow-subtle for body text (less performance impact)
- Added .phosphor-box-glow for larger elements

Now the glow will automatically match whatever color variant is active (green, amber, cyan, etc.) because --phosphor-glow is defined per variant in globals.css.
