<task type="auto">
  <name>Task 2: Create 6 monochrome color variants</name>
  <files>app/globals.css</files>
  <action>
In `app/globals.css`, after the existing `.theme-protheus` block (around line 1282), add 5 additional color variant classes. The existing green theme becomes the default, plus add amber, cyan, red, white, and orange variants.

Each variant uses a compound selector pattern: `.theme-protheus.variant-{color}` for light mode with `@variant dark` for dark mode.

Add these variant definitions:

```css
/* ═══════════════════════════════════════════════════════════════════════════
   PROTHEUS COLOR VARIANTS - Monochrome terminal palettes
   ═══════════════════════════════════════════════════════════════════════════ */

/* Green is the default (already defined in .theme-protheus) */
.theme-protheus.variant-green {
  /* Inherits from .theme-protheus - no overrides needed */
}

/* Amber variant - warm CRT phosphor */
.theme-protheus.variant-amber {
  --phosphor: oklch(0.85 0.18 85);
  --phosphor-dim: oklch(0.45 0.12 85);
  --phosphor-bright: oklch(0.95 0.22 85);
  --phosphor-glow: oklch(0.7 0.15 85 / 0.5);

  --foreground: oklch(0.15 0.05 85);
  --primary: oklch(0.15 0.05 85);
  --border: oklch(0.2 0.08 85);

  @variant dark {
    --foreground: oklch(0.85 0.18 85);
    --primary: oklch(0.85 0.18 85);
    --border: oklch(0.85 0.18 85);
  }
}

/* Cyan variant - cool blue-green terminal */
.theme-protheus.variant-cyan {
  --phosphor: oklch(0.85 0.15 195);
  --phosphor-dim: oklch(0.45 0.10 195);
  --phosphor-bright: oklch(0.95 0.18 195);
  --phosphor-glow: oklch(0.7 0.12 195 / 0.5);

  --foreground: oklch(0.15 0.05 195);
  --primary: oklch(0.15 0.05 195);
  --border: oklch(0.2 0.08 195);

  @variant dark {
    --foreground: oklch(0.85 0.15 195);
    --primary: oklch(0.85 0.15 195);
    --border: oklch(0.85 0.15 195);
  }
}

/* Red variant - alert/warning terminal */
.theme-protheus.variant-red {
  --phosphor: oklch(0.70 0.22 25);
  --phosphor-dim: oklch(0.40 0.15 25);
  --phosphor-bright: oklch(0.80 0.28 25);
  --phosphor-glow: oklch(0.60 0.18 25 / 0.5);

  --foreground: oklch(0.15 0.08 25);
  --primary: oklch(0.15 0.08 25);
  --border: oklch(0.2 0.10 25);

  @variant dark {
    --foreground: oklch(0.70 0.22 25);
    --primary: oklch(0.70 0.22 25);
    --border: oklch(0.70 0.22 25);
  }
}

/* White variant - high contrast monochrome */
.theme-protheus.variant-white {
  --phosphor: oklch(0.95 0 0);
  --phosphor-dim: oklch(0.60 0 0);
  --phosphor-bright: oklch(1.0 0 0);
  --phosphor-glow: oklch(0.85 0 0 / 0.5);

  --foreground: oklch(0.15 0 0);
  --primary: oklch(0.15 0 0);
  --border: oklch(0.25 0 0);

  @variant dark {
    --foreground: oklch(0.95 0 0);
    --primary: oklch(0.95 0 0);
    --border: oklch(0.95 0 0);
  }
}

/* Orange variant - vintage amber-orange */
.theme-protheus.variant-orange {
  --phosphor: oklch(0.80 0.20 55);
  --phosphor-dim: oklch(0.45 0.14 55);
  --phosphor-bright: oklch(0.90 0.25 55);
  --phosphor-glow: oklch(0.70 0.17 55 / 0.5);

  --foreground: oklch(0.15 0.06 55);
  --primary: oklch(0.15 0.06 55);
  --border: oklch(0.2 0.08 55);

  @variant dark {
    --foreground: oklch(0.80 0.20 55);
    --primary: oklch(0.80 0.20 55);
    --border: oklch(0.80 0.20 55);
  }
}
```

Color science notes:
- Green (hue 142): Classic phosphor green, existing
- Amber (hue 85): Warm golden-yellow phosphor
- Cyan (hue 195): Cool blue-green for cyberpunk feel
- Red (hue 25): Alert/warning aesthetic
- White (hue 0, chroma 0): Pure monochrome
- Orange (hue 55): Vintage warm terminal

All use oklch() for perceptual uniformity and consistent brightness across themes.
