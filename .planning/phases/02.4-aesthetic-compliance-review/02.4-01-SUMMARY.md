## Decisions Made

**CSS double border for terminal authenticity**
- Used `border-2 border-double` CSS classes instead of single border
- Matches authentic MS-DOS terminal double-line box drawing characters
- Maintains thin 1px aesthetic (double border uses 3px total: 1px + 1px gap + 1px)

**Framed variant uses rounded-full**
- Circular avatars match the double border's rounded shape
- Non-framed variants (default, cyphercn) use rounded-none with custom corner borders

**Preserved image-rendering-pixelated for cyphercn variant**
- CSS class remains valid for cyphercn variant image rendering
- Not related to variant naming - purely CSS property
